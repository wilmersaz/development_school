👣 Los 6 comandos que necesitas saber

Aprende los comandos esenciales de Git de forma simple y práctica

⚠️ ¡Importante!

¡No te asustes! Aunque se llamen "comandos", son súper simples. Es como aprender a usar WhatsApp - al principio parece complicado, pero después lo haces sin pensar.

👤 PRIMERO: "¡Hola Git, soy yo!"

Como poner tu nombre en WhatsApp para que todos sepan quién eres

git config --global user.name "Tu Nombre Completo"
git config --global user.email "tu@email.com"

¿Qué hace? Le dices a Git cómo te llamas y tu email. ¡SÚPER IMPORTANTE! Sin esto, Git no sabe quién hizo los cambios.

📝 Ejemplo real:

git config --global user.name "María García"
git config --global user.email "maria.garcia@gmail.com"

¡Una sola vez! Solo necesitas hacer esto la primera vez que uses Git en tu computadora.

🔍 ¿Quieres verificar que funcionó?

Usa estos comandos para ver tu configuración:

git config user.name
git config user.email

Si ves tu nombre y email, ¡perfecto! Git ya te conoce. 😊

🏠 Comando 1: "¡Git, cuida esta carpeta!"

Como decirle a un guardia de seguridad "vigila esta casa"

git init

¿Qué hace? Convierte cualquier carpeta normal en una carpeta "mágica" que Git puede vigilar y guardar versiones.

🎯 ¿Cuándo usarlo?

Situación: Tienes una carpeta con tus documentos de trabajo

Problema: Git no la conoce, no puede cuidarla

Solución: git init

Resultado: ¡Ahora Git puede guardar versiones de todo lo que está ahí! ✨

🔍 Comando 2: "¿Qué está pasando?"

Como preguntar "¿hay mensajes nuevos?" en WhatsApp

git status

¿Qué hace? Te dice si hay archivos nuevos o cambios que Git no ha guardado todavía.

➕ Comando 3: "Quiero guardar esto"

Como seleccionar las fotos que quieres subir a Instagram

git add archivo.txt
git add .

¿Qué hace? Le dices a Git "estos archivos son importantes, prepáralos para guardar". El punto (.) significa "todos los archivos".

💾 Comando 4: "¡Guárdalo ahora!"

Como hacer clic en "Publicar" en Facebook

git commit -m "Agregué mi foto de perfil"

¿Qué hace? Guarda una "foto" de tu proyecto con una nota explicando qué cambiaste. ¡Siempre pon una nota descriptiva!

☁️ Comando 5: "Súbelo a internet"

Como subir un video a YouTube

git push

¿Qué hace? Toma todos tus cambios guardados y los sube a GitHub para que otros los vean (o para tener respaldo en la nube).

⬇️ Comando 6: "Descargar lo nuevo"

Como actualizar WhatsApp para ver los mensajes nuevos

git pull

¿Qué hace? Si otras personas han hecho cambios, este comando los descarga a tu computadora.

🎭 ¡Veamos esto en acción! (Ejemplo completo)

Situación: Estás trabajando en una presentación para la universidad

Paso a paso:

0

Preparas la carpeta para Git:

git init

"Git, cuida esta carpeta"

1

Revisas qué has cambiado:

git status

"Cambié 2 archivos: presentacion.pptx y notas.txt"

2

Seleccionas lo que quieres guardar:

git add .

"Ok, guarda todo lo que cambié"

3

Guardas con una nota:

git commit -m "Agregué slides sobre historia"
4

Lo subes a GitHub:

git push

¡Ya está en internet! 🎉

Resultado: Tu presentación está guardada PARA SIEMPRE, tienes respaldo en la nube, y puedes acceder desde cualquier computadora.
