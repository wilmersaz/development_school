🏠 Ejemplos del día a día

Entiende Git con comparaciones que usas todos los días

📚 Como escribir un libro en equipo

Imagina que tú y 4 amigos están escribiendo un libro juntos. Sin Git, sería un caos:

😱 SIN Git

  • • Enviar emails con "mi_parte_del_libro.doc"
  • • "¿Cuál es la versión más nueva?"
  • • Dos personas cambian lo mismo
  • • Se pierden cambios importantes

😊 CON Git

  • • Todos trabajan en el mismo documento
  • • Git combina los cambios automáticamente
  • • Nunca se pierde nada
  • • Puedes ver quién cambió qué y cuándo

🏗️ Como construir una casa

15

30
Día 30: Se pone el techo (Git guarda otra "foto")
?
¿Algo salió mal? ¡Puedes volver a cualquier día anterior!

🌟 Ejemplo de la vida real: Tu CV

Situación: Estás mejorando tu currículum para diferentes trabajos.

1

Versión base: Tu CV actual

Git toma una "foto" y la llama "CV básico"

2

Para trabajo en marketing: Agregas experiencia en redes sociales

Git toma otra "foto" y la llama "CV marketing"

3

Para trabajo técnico: Agregas cursos de computación

Git toma otra "foto" y la llama "CV técnico"

✨

¡Magia! Puedes cambiar entre versiones cuando quieras

Git te permite volver a cualquier versión en segundos

⚠️ ¡Importante!

¿Por qué es importante?
En el mundo actual, TODOS trabajamos con documentos digitales: presentaciones, informes, proyectos universitarios, etc. Git te ayuda a no perder NUNCA tu trabajo y a colaborar sin problemas.
