🎯 Paso 1: Crear cuenta en GitHub (GRATIS)

Es como crear una cuenta de Facebook, pero para tus proyectos:

  1. 1 Ve a github.com
  2. 2 Haz clic en "Sign up" (Registrarse)
  3. 3 Elige un nombre de usuario (como @tu_nombre)
  4. 4 Pon tu email y una contraseña
  5. 5 ¡Listo! Ya tienes tu "casa en internet" 🏠

💻 Paso 2: Instalar Git en tu computadora

¿Tienes Windows?

  • 1 Ve a git-scm.com
  • 2 Haz clic en "Download" (Descargar)
  • 3 Instala como cualquier programa normal
  • 4 ¡Ya está! Git ahora vive en tu computadora

⚠️ ¡Importante!

¿Mac o Linux? ¡Probablemente ya tienes Git instalado! Es como si viniera incluido en la caja. 📦

🎉 ¡Felicidades! Ya tienes lo básico

Con estos dos pasos ya puedes:

✅ Guardar versiones de tus archivos
✅ Subir tus proyectos a internet
✅ Colaborar con otras personas
✅ Impresionar a tus jefes/profesores 😎

💡 Tips para empezar bien

📚 Para estudiar

Usa Git para guardar versiones de tus trabajos, tesis o proyectos universitarios. ¡Nunca más perderás tu trabajo!

💼 Para trabajar

Organiza documentos de trabajo, presentaciones y colabora con tu equipo de forma profesional.

🎨 Para proyectos personales

Guarda versiones de tus fotos editadas, documentos personales o cualquier proyecto creativo.

👥 Para colaborar

Trabaja en equipo sin los típicos problemas de "¿cuál es la versión más nueva?" o archivos perdidos.
