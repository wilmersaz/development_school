🎯 ¡Hora de practicar! (Súper simple)

Crea tu primer proyecto Git paso a paso

🎮 Tu primera misión

Vamos a crear tu primer proyecto en Git. Es como crear tu primer post en Instagram - ¡nervioso pero emocionante!

🚀 Misión: "Mi primer proyecto"

0

¡PRIMERO! Preséntate a Git

Abre la terminal y dile a Git quién eres (una sola vez en tu vida)

git config --global user.name "Tu Nombre"
git config --global user.email "tu@email.com"
1

Crea una carpeta en tu escritorio

Llámala "mi-primer-proyecto" (sin espacios, usa guiones)

2

Crea un archivo de texto dentro

Llámalo "readme.txt" y escribe: "¡Hola mundo! Este es mi primer proyecto con Git"

3

Abre la "terminal" o "símbolo del sistema"

En Windows: busca "cmd" en el menú inicio

4

Ve a tu carpeta

cd Desktop/mi-primer-proyecto

(Como navegar a una carpeta en el explorador, pero con texto)

5

¡Inicializa Git!

git init

Le dices a Git: "¡Cuida esta carpeta por favor!"

6

Usa los comandos que aprendiste

git add .
git commit -m "Mi primer commit"

¡Acabas de guardar tu primer "snapshot"!

🎉 ¡LO LOGRASTE!

¿Qué acabas de hacer?

✅ Creaste tu primer repositorio de Git
✅ Guardaste tu primer "commit" (versión)
✅ Ya sabes usar Git básico
✅ ¡Eres oficialmente un usuario de Git! 🎊

🚀 ¿Qué sigue? (Ideas para seguir practicando)

📁 Crea más archivos:

Agrega una foto, un documento de Word, lo que quieras

📸 Haz más commits:

Cada vez que cambies algo, guarda una nueva versión

☁️ Sube a GitHub:

Crea un repositorio en github.com y sube tu proyecto

👥 Invita a un amigo:

Compartan un proyecto y trabajen juntos

⚠️ ¡Recuerda!

Git no es solo para programadores. Puedes usar Git para CUALQUIER proyecto: documentos de trabajo, fotos de familia organizadas, apuntes de universidad, ¡lo que se te ocurra!

🌐 Conectar con GitHub

Sube tu proyecto a internet para acceder desde cualquier lugar.

git remote add origin https://github.com/tu-usuario/mi-primer-proyecto.git
git push -u origin main

🔄 Trabajar en colaboración

Invita a otros a tu proyecto y trabajen juntos.

git clone https://github.com/usuario/proyecto.git
git pull origin main
