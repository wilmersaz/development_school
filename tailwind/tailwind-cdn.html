🌊 Integrar Tailwind CSS con CDN

La forma más rápida de empezar con Tailwind CSS

🚀¿Qué es Tailwind CSS CDN?

El CDN de Tailwind CSS es una versión simplificada que incluye todas las clases utilitarias predefinidas, perfecta para prototipado rápido y proyectos sin herramientas de build.

✅ Ventajas del CDN

  • • Setup instantáneo
  • • No requiere configuración
  • • Perfecto para prototipado
  • • Todas las clases disponibles
  • • Ideal para aprender

⚠️ Limitaciones del CDN

  • • Archivo más grande (~3MB)
  • • No personalizable
  • • Sin PurgeCSS automático
  • • No apto para producción

📝Integración en 3 Pasos

1 Obtener el CDN

Ve a tailwindcss.com o usa el CDN oficial más reciente:

📄 index.html 
<script src="https://cdn.tailwindcss.com"></script>

2 Estructura HTML Básica

Crea un archivo HTML con la estructura básica:

📄 index.html 
<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi sitio con Tailwind CSS</title>
    <!-- Tailwind CSS CDN -->
    <script src="https://cdn.tailwindcss.com"></script>
  </head>
  <body>
    <!-- Tu contenido aquí -->
    <h1 class="text-4xl font-bold text-blue-600">¡Hola Tailwind!</h1>
  </body>
</html>

3 ¡Empezar a usar clases!

Tailwind funciona con clases utilitarias que describes directamente en el HTML:

📄 index.html 
<div class="bg-blue-500 text-white p-6 rounded-lg shadow-lg">
  <h2 class="text-2xl font-bold mb-4">¡Tailwind es genial!</h2>
  <p class="text-blue-100 mb-4">
    Esto es una tarjeta creada con clases utilitarias.
  </p>
  <button class="bg-white text-blue-500 px-4 py-2 rounded hover:bg-gray-100 transition-colors">
    Botón de Acción
  </button>
</div>

🎨Ejemplo Completo

Aquí tienes una página completa usando Tailwind con CDN:

📄 index.html 
<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primera Página con Tailwind</title>
    <script src="https://cdn.tailwindcss.com"></script>
  </head>
  <body class="bg-gray-100 min-h-screen py-8">
    <div class="max-w-md mx-auto">
      <div class="bg-white rounded-xl shadow-lg overflow-hidden">
        <div class="bg-gradient-to-r from-blue-500 to-purple-500 p-6 text-white text-center">
          <div class="w-20 h-20 bg-white rounded-full mx-auto mb-4 flex items-center justify-center">
            <span class="text-3xl">🚀</span>
          </div>
          <h1 class="text-2xl font-bold">¡Hola Tailwind!</h1>
          <p class="text-blue-100">Mi primera tarjeta</p>
        </div>
        <div class="p-6">
          <p class="text-gray-600 mb-4">
            Esta tarjeta fue creada usando únicamente clases utilitarias de Tailwind CSS.
          </p>
          <div class="flex space-x-4">
            <button class="bg-blue-500 text-white px-4 py-2 rounded-lg hover:bg-blue-600 transition-colors">
              Primario
            </button>
            <button class="border border-gray-300 text-gray-700 px-4 py-2 rounded-lg hover:bg-gray-50 transition-colors">
              Secundario
            </button>
          </div>
        </div>
      </div>
    </div>
  </body>
</html>

⚙️Configuración Avanzada (Opcional)

Puedes personalizar Tailwind CDN añadiendo tu propia configuración:

📄 index.html 
    <script src="https://cdn.tailwindcss.com"></script>
    <script>
      tailwind.config = {
        theme: {
          extend: {
            colors: {
              'custom-blue': '#1e40af',
              'custom-purple': '#7c3aed'
            },
            fontFamily: {
              'custom': ['Inter', 'sans-serif']
            }
          }
        }
      }
    </script>

⚠️ Nota: Esta configuración solo funciona con el CDN. Para proyectos reales, considera usar la instalación completa de Tailwind.

⚖️Tailwind CDN vs Bootstrap CDN

Aspecto Tailwind CDN Bootstrap CDN
🏷️ Integración <script> tag <link> tag
📦 Tamaño ~3MB (todas las clases) ~20KB (minificado)
🎨 Filosofía Utility-first Component-based
⚡ JavaScript No incluido Incluido
🎯 Uso ideal Diseños únicos Prototipos rápidos
