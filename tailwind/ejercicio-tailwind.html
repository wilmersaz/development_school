🎯 Ejercicio Práctico: Tarjeta Tailwind

Crea una tarjeta de presentación con utilidades de Tailwind CSS

🎯 Objetivo del Ejercicio

Vamos a crear una tarjeta de presentación personal utilizando únicamente clases utilitarias de Tailwind CSS. El enfoque será construir cada elemento desde cero usando utilidades.

📋 Requisitos:

  • ✅ Usar enfoque utility-first
  • ✅ Aplicar sistema de colores de Tailwind
  • ✅ Implementar diseño responsive
  • ✅ Usar flexbox y grid
  • ✅ Aplicar efectos hover y transiciones
  • ✅ Usar al menos 20 clases de Tailwind

📝Guía Paso a Paso

1 Estructura HTML y CDN

Configura el HTML básico con Tailwind CDN:

📄 index.html 
<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Tarjeta con Tailwind</title>
    <script src="https://cdn.tailwindcss.com"></script>
  </head>
  <body class="bg-gray-100 min-h-screen py-8">
    <!-- Aquí irá tu tarjeta -->
  </body>
</html>

2 Contenedor y Estructura Base

Crea el contenedor principal y la estructura de la tarjeta:

📄 index.html 
<body class="bg-gray-100 min-h-screen py-8">
  <div class="max-w-sm mx-auto">
    <div class="bg-white rounded-xl shadow-xl overflow-hidden">
      
      <!-- Header -->
      <div class="bg-gradient-to-r from-cyan-500 to-blue-500 p-8 text-white text-center">
        <!-- Avatar y título aquí -->
      </div>
      
      <!-- Body -->
      <div class="p-6">
        <!-- Contenido aquí -->
      </div>
      
    </div>
  </div>
</body>

⚖️ Comparación: Bootstrap vs Tailwind

Ahora que has creado tarjetas con ambos frameworks, nota las diferencias:

🅱️ Enfoque Bootstrap

.card .card-header .card-body

Componentes predefinidos

.btn .btn-primary .btn-lg

Variaciones de componentes

.container .row .col

Sistema de grid estructurado

✨ Ventaja: Componentes listos, desarrollo rápido

🌊 Enfoque Tailwind

.bg-white .rounded-xl .shadow-xl

Utilidades atómicas

.bg-gradient-to-r .from-cyan-500

Control granular de estilos

.grid .grid-cols-3 .gap-4

Layout flexible con utilidades

✨ Ventaja: Control total, diseños únicos

🚀 Desafíos Extra

🎖️ Nivel Intermedio

  • ✨ Agrega modo dark/light toggle
  • ✨ Crea animaciones personalizadas
  • ✨ Implementa un slider de proyectos
  • ✨ Añade formulario de contacto responsive

🏆 Nivel Avanzado

  • 🔥 Crea configuración custom de Tailwind
  • 🔥 Implementa lazy loading para imágenes
  • 🔥 Añade micro-interacciones avanzadas
  • 🔥 Optimiza para performance

💡 Tip Pro: Combina ambos enfoques según el proyecto. Bootstrap para prototipos rápidos, Tailwind para diseños únicos.
