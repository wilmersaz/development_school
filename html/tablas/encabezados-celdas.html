🎯 Encabezados y Celdas

Domina el uso correcto de <th> y <td> para crear tablas profesionales

⚖️ <th> vs <td>: ¿Cuándo usar cada uno?

🏷️ <th> - Table Header

✅ Cuándo usar <th>:

  • ▶ Títulos de columnas
  • ▶ Títulos de filas
  • ▶ Categorías principales
  • ▶ Encabezados de sección
Características por defecto:
  • • Negrita: font-weight: bold
  • • Centrado: text-align: center
  • • Accesible: Lectores de pantalla

📦 <td> - Table Data

✅ Cuándo usar <td>:

  • ▶ Datos específicos
  • ▶ Información detallada
  • ▶ Valores numéricos
  • ▶ Contenido variable
Características por defecto:
  • • Texto normal: font-weight: normal
  • • Izquierda: text-align: left
  • • Contenido: Cualquier HTML válido

💡 Ejemplos Prácticos

👥 Tabla de Empleados

📝 Código HTML

📄 index.html 
<table>
  <tr>
      <th>Nombre</th>
      <th>Cargo</th>
      <th>Salario</th>
  </tr>
  <tr>
      <td>Ana García</td>
      <td>Desarrolladora</td>
      <td>$3,500</td>
  </tr>
</table>

🎯 Resultado

Nombre Cargo Salario
Ana García Desarrolladora $3,500
Carlos López Diseñador $3,200

Nota: Los encabezados (<th>) están en negrita y centrados automáticamente

📊 Encabezados de Fila

📝 Código HTML

📄 index.html 
// <th> también puede ir en la primera columna
<table>
  <tr>
    <th>Producto</th>
    <td>Laptop HP</td>
  </tr>
  <tr>
    <th>Precio</th>
    <td>$899</td>
  </tr>
  <tr>
    <th>Stock</th>
    <td>15 unidades</td>
  </tr>
</table>

🎯 Resultado

Producto Laptop HP
Precio $899
Stock 15 unidades

Flexibilidad: Los <th> pueden ser encabezados de fila o columna

🎯 Atributo Scope para Accesibilidad

El atributo scope mejora la accesibilidad al especificar si un encabezado se refiere a una columna o fila.

📋 Valores de Scope

scope="col"

El encabezado se refiere a toda la columna

scope="row"

El encabezado se refiere a toda la fila

scope="colgroup"

El encabezado se refiere a un grupo de columnas

scope="rowgroup"

El encabezado se refiere a un grupo de filas

💻 Ejemplo con Scope

<table>
<tr>
<th scope="col">Nombre</th>
<th scope="col">Edad</th>
</tr>
<tr>
<td>María</td>
<td>28</td>
</tr>
</table>

🌟 Beneficios:

  • • Mejor navegación con lectores de pantalla
  • • Comprensión clara de la estructura
  • • Cumple estándares de accesibilidad
  • • Mejora la experiencia del usuario

📦 Tipos de Contenido en Celdas

📝 Texto Simple

<td>Juan Pérez</td>

Texto plano, nombres, descripciones

💰 Números

<td>$1,250.50</td>

Precios, cantidades, porcentajes

🔗 Enlaces

<td><a href="#">Ver más</a></td>

Links, botones, acciones

🖼️ Imágenes

<td><img src="avatar.jpg"></td>

Fotos, íconos, gráficos

🎨 HTML Complejo

<td><span class="badge">Activo</span></td>

Elementos con estilos, badges

📅 Fechas

<td>15/08/2025</td>

Fechas, horarios, timestamps

🎯 Ejemplo Combinado

Usuario Avatar Estado Último Acceso Acción
Ana García
A
Activo 19/08/2025

📚 Resumen del Tema

🏷️ <th> - Encabezados

  • • Para títulos de columnas y filas
  • • Negrita y centrado por defecto
  • • Usar atributo scope para accesibilidad
  • • Mejora la experiencia del usuario

📦 <td> - Datos

  • • Para el contenido real de la tabla
  • • Acepta cualquier contenido HTML
  • • Texto normal y alineado a la izquierda
  • • Flexible para diferentes tipos de datos
