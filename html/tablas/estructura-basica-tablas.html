🏗️ Estructura Básica de Tablas

Aprende los elementos fundamentales para crear una tabla HTML

🔍 Anatomía de una Tabla

📋 Elementos Principales

<table> - Contenedor Principal

Define el inicio y fin de la tabla completa

<tr> - Fila de Tabla (Table Row)

Representa cada fila horizontal en la tabla

<td> - Celda de Datos (Table Data)

Contiene los datos individuales de cada celda

<th> - Celda de Encabezado (Table Header)

Define los títulos o encabezados de columnas/filas

🎯 Tabla Visual

<th> Nombre <th> Edad
<td> Ana <td> 25
<td> Luis <td> 30

Estructura: table → tr → th/td

💻 Tu Primera Tabla

📝 Código HTML

// Estructura básica de una tabla
<table>
<tr>
<th>Producto</th>
<th>Precio</th>
<th>Stock</th>
</tr>
<tr>
<td>Laptop</td>
<td>$800</td>
<td>15</td>
</tr>
<tr>
<td>Mouse</td>
<td>$25</td>
<td>50</td>
</tr>
</table>

🎯 Resultado

Producto Precio Stock
Laptop $800 15
Mouse $25 50

¡Perfecto! Tu tabla muestra los datos de manera organizada y fácil de leer.

⚡ Reglas Importantes

📏 Estructura Obligatoria

🎯 Jerarquía Correcta

La estructura debe seguir este orden:

table → tr → (th o td)

⚖️ Columnas Consistentes

Todas las filas deben tener el mismo número de columnas

🏷️ Encabezados Claros

Usa <th> para encabezados, <td> para datos

❌ Errores Comunes

🚫 Etiquetas Mal Anidadas

❌ <table><td>Datos</td></table>
✅ <table><tr><td>Datos</td></tr></table>

🚫 Filas Desiguales

No mezclar filas con diferentes números de celdas sin usar colspan

🚫 Etiquetas sin Cerrar

Siempre cerrar todas las etiquetas: <tr>, <td>, <th>

🎯 Ejercicio Práctico

📚 Crea tu Tabla de Estudiantes

Intenta crear una tabla que muestre información de estudiantes con las siguientes columnas:

  • • Nombre: Nombre completo del estudiante
  • • Carrera: Programa de estudios
  • • Semestre: Nivel académico actual
  • • Promedio: Calificación promedio

💡 Pista de Código

// Estructura sugerida
<table>
<tr>
<th>Nombre</th>
<th>Carrera</th>
// Completa las demás columnas...
</tr>
// Agrega filas de datos aquí...
</table>

🎯 Resultado Esperado

Nombre Carrera Semestre Promedio
Carlos Mendez Ingeniería 5° 8.5
Sofia Rodriguez Medicina 3° 9.2
