🏗️ Estructura Avanzada de Tablas

Aprende a usar <thead>, <tbody> y <tfoot> para tablas profesionales

🎯 ¿Por qué Estructura Avanzada?

🤔 El Problema

Las tablas básicas con solo <tr> y <td> funcionan, pero tienen limitaciones:

❌ Difícil aplicar estilos específicos por sección

❌ Los navegadores no pueden optimizar el renderizado

❌ Accesibilidad limitada para lectores de pantalla

❌ JavaScript no puede manipular secciones fácilmente

✅ La Solución

HTML5 proporciona etiquetas semánticas que dividen la tabla en secciones lógicas:

✅ <thead> - Encabezados de la tabla

✅ <tbody> - Cuerpo principal con datos

✅ <tfoot> - Pie con resúmenes/totales

🎯 Mejor organización y funcionalidad

<thead> - Sección de Encabezados

📋 Descripción

<thead> agrupa las filas de encabezado de una tabla. Generalmente contiene información como títulos de columnas.

🎯 Características:

  • • Debe aparecer antes de <tbody> y <tfoot>
  • • Puede contener una o más filas (<tr>)
  • • Ideal para encabezados complejos (multi-nivel)
  • • Los navegadores pueden repetirlo al imprimir páginas
  • • Fácil de estilizar con CSS

💻 Ejemplo de Uso

// Estructura con thead
<table>
<thead>
<tr>
<th>Producto</th>
<th>Precio</th>
<th>Stock</th>
</tr>
</thead>
// tbody y tfoot van aquí...
</table>

Consejo: Usa <thead> incluso si solo tienes una fila de encabezados.

<tbody> - Cuerpo de la Tabla

📦 Descripción

<tbody> contiene el cuerpo principal de la tabla con todos los datos reales.

🎯 Características:

  • • Contiene las filas con datos reales
  • • Puede tener múltiples <tbody> por tabla
  • • Útil para agrupar datos relacionados
  • • Permite estilos alternativos (filas pares/impares)
  • • Facilita el scroll independiente del header

¿Sabías que? Si no incluyes <tbody>, el navegador lo crea automáticamente.

🎨 Múltiples tbody

// Múltiples tbody para agrupar datos
<table>
<thead>...</thead>
<tbody class="grupo-a">
<tr>...</tr>
<tr>...</tr>
</tbody>
<tbody class="grupo-b">
<tr>...</tr>
<tr>...</tr>
</tbody>
</table>

Ventaja: Cada grupo puede tener estilos diferentes o comportamiento JavaScript específico.

<tfoot> - Pie de la Tabla

🦶 Descripción

<tfoot> define el pie de la tabla, típicamente usado para totales, resúmenes o notas finales.

🎯 Usos Comunes:

  • • Totales: Sumas de columnas numéricas
  • • Promedios: Cálculos estadísticos
  • • Notas: Información adicional
  • • Paginación: Controles de navegación
  • • Acciones: Botones de acciones masivas

Orden HTML: tfoot puede ir antes de tbody, pero se renderizará al final.

💰 Ejemplo de Totales

// tfoot con totales
<table>
<thead>...</thead>
<tbody>...</tbody>
<tfoot>
<tr>
<td>Total</td>
<td>$2,450</td>
<td>125 items</td>
</tr>
</tfoot>
</table>

Tip CSS: Puedes fijar el tfoot al final con position: sticky.

🎯 Ejemplo Completo: Tabla de Ventas

📝 Código HTML Completo

// Tabla completa con todas las secciones
<table class="ventas-tabla">
// ENCABEZADO
<thead>
<tr>
<th scope="col">Producto</th>
<th scope="col">Cantidad</th>
<th scope="col">Precio Unit.</th>
<th scope="col">Subtotal</th>
</tr>
</thead>
// CUERPO - DATOS
<tbody>
<tr>
<td>Laptop HP</td>
<td>2</td>
<td>$899</td>
<td>$1,798</td>
</tr>
<tr>
<td>Mouse Logitech</td>
<td>5</td>
<td>$45</td>
<td>$225</td>
</tr>
</tbody>
// PIE - TOTALES
<tfoot>
<tr>
<td colspan="3"><strong>Total General</strong></td>
<td><strong>$2,023</strong></td>
</tr>
</tfoot>
</table>

🎨 Resultado Visual

Producto Cant. Precio Subtotal
Laptop HP 2 $899 $1,798
Mouse Logitech 5 $45 $225
Total General $2,023
THEAD: Encabezados con fondo azul
TBODY: Datos alternando colores
TFOOT: Total con fondo naranja

Fácil manipulación de secciones

🖨️ Impresión

Los navegadores pueden repetir thead en cada página impresa

📱 Responsivo

Mejor control del comportamiento en dispositivos móviles

🔍 SEO

Los buscadores entienden mejor la estructura de datos

📚 Resumen de Estructura Avanzada

📋

<thead>

Agrupa los encabezados de la tabla. Ideal para títulos de columnas y estructura del header.

📦

<tbody>

Contiene el cuerpo principal con datos. Puede tener múltiples tbody para agrupar.

🦶

<tfoot>

Define el pie con totales, resúmenes o información adicional de la tabla.

Orden recomendado: thead → tbody → tfoot
