🚀 Para Principiantes

✅ Buenas Prácticas

  • • Siempre cierra las etiquetas que abres
  • • Usa indentación para mayor claridad
  • • Los atributos van entre comillas
  • • Un archivo HTML bien formado empieza con <!DOCTYPE html>

❌ Errores Comunes

  • • Olvidar cerrar etiquetas: <p>texto (falta </p>)
  • • Anidar incorrectamente: <p><div></p></div>
  • • Atributos sin comillas: <img src=foto.jpg>
  • • Usar etiquetas inexistentes: <texto>

⌨️ Atajos de Teclado

En el Editor:

Limpiar todo Ctrl+A → Delete
Deshacer Ctrl+Z
Rehacer Ctrl+Y
Buscar Ctrl+F

HTML Rápido:

<h1> + Tab = <h1></h1>
<p> + Tab = <p></p>
<a> + Tab = <a href=""></a>
<img> + Tab = <img src="" alt="">
