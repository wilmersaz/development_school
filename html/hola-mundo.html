📄 ¿Qué es index.html?

🏠 La Página Principal

index.html es el nombre estándar para la página principal de un sitio web. Es como la "puerta de entrada" de tu sitio.

¿Por qué "index"?

  • • Tradición: Viene de los servidores web antiguos
  • • Automático: Los navegadores lo buscan primero
  • • Estándar: Todos los servidores lo reconocen
  • • Limpio: URLs sin nombre de archivo (ejemplo.com/)

Equivalencias:

  • • misitioweb.com = misitioweb.com/index.html
  • • ejemplo.com/productos/ = ejemplo.com/productos/index.html

  • • <ul>, <li> - Lista con viñetas
4

Guardar el Archivo

💾 Pasos para guardar:

  1. 1

    Presiona Ctrl+S (o Cmd+S en Mac)

    O ve al menú Archivo → Guardar

  2. 2

    Elige una carpeta

    Crea una carpeta nueva llamada "mi-primer-sitio"

  3. 3

    Nombra el archivo: index.html

    ¡IMPORTANTE! Debe terminar en .html

  4. 4

    Cambia el tipo a "Todos los archivos" o "HTML"

    Para evitar que se guarde como .txt

⚠️ Errores Comunes:
  • • Guardar como "index.html.txt" ❌
  • • Usar espacios en el nombre ❌
  • • Olvidar la extensión .html ❌
  • • Usar caracteres especiales (ñ, acentos) ❌
✅ Correcto:
  • • index.html ✅
  • • mi-pagina.html ✅
  • • home.html ✅
  • • inicio.html ✅
5

Abrir en el Navegador

🌐 Métodos para abrir:

Método 1: Doble clic

Busca tu archivo index.html en el explorador y haz doble clic

Método 2: Arrastrar y soltar

Arrastra el archivo desde el explorador hasta una ventana del navegador

Método 3: Desde el navegador

Ctrl+O en el navegador y busca tu archivo
